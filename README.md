# AI Bootcamp — Skills & Evals

Submissions from the Transcarent Product & Design AI Uplevel Workshop. Each file is a skill, eval/rubric, or command built during bootcamp and contributed by a team member.

## How to contribute

Use the submission app: **[bootcamp-skills-app.netlify.app](https://bootcamp-skills-app.netlify.app)**

Fill out the form and your file will land here automatically. No GitHub account needed.

## Structure

```
submissions/
  skills/      — SKILL.md files Claude Code picks up automatically by context
  evals/       — Rubric checks and evaluator files for recurring outputs
  commands/    — Slash commands invoked explicitly by name
```

Each file includes contributor name, team, description, and the full file contents.
